`Button` follows the [WAI-ARIA - Button Pattern](https://www.w3.org/WAI/ARIA/apg/patterns/button/)for accessibility.

## ARIA Role and Attributes

| Component | Roles and Attributes | Usage                          |
| --------- | -------------------- | ------------------------------ |
| `Button`  | `role="button"`      | Indicates that it is a button. |
